# Ashdod Tech — אפיון מוצר ראשוני

## חזון

Ashdod Tech היא מערכת למידה עירונית לתלמידים, בני נוער ומורים באשדוד. המערכת משלבת קורסים עירוניים, קורסים מתוך Campus IL ומסלול ייחודי שבו ילדים ובני נוער יכולים להציע, לבנות ולהעביר קורסים בליווי מקצועי.

## קהלי יעד

1. תלמידים
2. בני נוער
3. מורים
4. מנהלי תוכן ואדמינים עירוניים

## Campus IL Integration

בשלב ראשון המערכת תשמור רשומת קורס חיצוני:

- מזהה משתמש
- שם הקורס
- מקור: Campus IL
- קישור חיצוני
- תאריך התחלה
- סטטוס: נפתח / בלמידה / הושלם ידנית / תעודה נוספה
- קובץ תעודה אופציונלי

התקדמות אוטומטית מתוך Campus IL תדרוש חיבור API או שיתוף פעולה רשמי.

## MVP מומלץ

4. שמירת קורס חיצוני בהיסטוריית המשתמש
5. טאב יוצרים קורס
6. טופס הגשת הצעה
7. דשבורד אישי בסיסי
