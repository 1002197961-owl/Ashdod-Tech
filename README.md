# Ashdod Tech

פלטפורמת למידה עירונית לתלמידים, בני נוער ומורים באשדוד.

המיזם משלב:

- קטלוג קורסים עירוני
- קורסים חיצוניים מתוך Campus IL
- היסטוריית למידה אישית
- מסלול ייחודי שבו ילדים ובני נוער מציעים, בונים ומעבירים קורסים בליווי מקצועי
- ממשק RTL בעברית
- עיצוב Gov-Tech מודרני: מקצועי, נקי, עירוני וחדשני

## Tech Stack

- Next.js 15
- React 19
- TypeScript
- Tailwind CSS
- App Router

## Main Screens

- דף בית
- קטלוג קורסים
- קורסי Campus IL
- יוצרים קורס
- הקורסים שלי
- דשבורד תלמיד / מורה
- תעודות והיסטוריית למידה

## Development

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Product Direction

Ashdod Tech is not only a course catalog. It is a civic learning ecosystem where the city, teachers, students and youth creators can build learning opportunities together.
